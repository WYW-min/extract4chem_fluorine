# 4.2常⻅错误预警

# 1. 数值单位混淆：

◦ 错误：拉伸强度标注为 "150" (⽆单位) 或 "150 GPa" (数量级错误，应为 MPa)。

◦ 对策：建⽴常⻅物理量范围阈值检查（如Tg通常在 $1 0 0 { - } 5 0 0 ^ { \circ } \mathsf C ,$ ）。

# 2. 层级结构错误：

◦ 错误：将 测试条件 直接放在根节点，⽽⾮嵌套在具体的 性质 对象中。

◦ 对策：严格遵循JSON嵌套结构。

# 3. 多样本混淆：

◦ 错误：在⼀篇⽂献包含多个聚合物（如PI-1,PI-2,PI-3）时，将其属性合并到⼀个对象中。

◦ 对策：必须在 聚合物 数组中创建多个独⽴对象，分别对应PI-1,PI-2等。
